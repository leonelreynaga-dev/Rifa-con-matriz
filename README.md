# Rifa en C++ (Proyecto Educativo)

## Descripción
Este proyecto simula una rifa de 100 números organizada en una matriz 10x10. 
Permite ver los números disponibles, comprar uno y restaurar un número comprado.

Se desarrolló con fines educativos para practicar:
- Manejo de arrays bidimensionales
- Menús interactivos
- Validación de datos de entrada
- Uso de colores en consola
